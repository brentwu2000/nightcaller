## 概述

### 遊戲簡介

阿瓦隆（The Resistance: Avalon）是一款社交推理桌遊。5-10 人，每局約 30-60 分鐘。好人陣營（亞瑟的騎士）vs 邪惡陣營（莫德雷德的爪牙），透過任務提名、投票、秘密任務執行決定勝負。

### 產品定位

**單裝置離線語音主持工具 + 計分板** — 一台設備放桌上，語音主持夜晚流程、追蹤任務結果、播報勝負。

玩家用**實體角色牌**分配角色，面對面進行投票和討論，App **不管每個玩家是誰、不追蹤個人角色**。

### 使用模式

```
設備放桌上當主持人：
  遊戲設定（人數 / 角色分佈配置）
  → 夜晚：TTS 語音引導閉眼睜眼流程（玩家用實體角色牌）
  → 白天：玩家面對面討論、組隊、投票（App 不介入）
  → 組織者回報任務結果（成功/失敗）
  → App 更新計分板、判斷勝負
  → 結算：勝負播報 + 刺殺梅林（若觸發）
```

### 不包含的功能

- ~~數位角色分配~~ — 用實體角色牌
- ~~傳閱查看角色~~ — 用實體角色牌
- ~~數位投票系統~~ — 面對面舉手/投票牌
- ~~秘密任務出牌~~ — 用實體任務牌
- ~~追蹤每個玩家的角色~~ — App 不知道誰是誰
- ~~隊長提名操作~~ — 面對面口頭指定
- 線上房間 / 多設備同步
- 後端伺服器 / 帳號系統

## 1. 遊戲規則摘要

### 1.1 陣營與勝利條件

| 陣營 | 勝利條件 |
|------|----------|
| 好人陣營 | 3 次任務成功 + 梅林未被刺殺 |
| 邪惡陣營 | 3 次任務失敗 / 5 次連續否決 / 刺殺梅林成功 |

### 1.2 角色說明

#### 好人陣營

| 角色 | 夜晚資訊 |
|------|----------|
| 梅林 (Merlin) | 睜眼看見所有邪惡成員（莫德雷德除外） |
| 派西維爾 (Percival) | 睜眼看見梅林和莫甘娜（無法分辨） |
| 好人騎士 (Loyal Servant) | 無特殊資訊 |

#### 邪惡陣營

| 角色 | 夜晚資訊 |
|------|----------|
| 刺客 (Assassin) | 與其他邪惡互認 |
| 莫甘娜 (Morgana) | 與其他邪惡互認；派西維爾看見她 |
| 莫德雷德 (Mordred) | 與其他邪惡互認；梅林看不見他 |
| 奧伯倫 (Oberon) | 不認識其他邪惡；其他邪惡也不認識他 |
| 邪惡爪牙 (Minion) | 與其他邪惡互認（奧伯倫除外） |

### 1.3 不同人數的配置

| 玩家數 | 好人 | 邪惡 | 任務人數（第 1-5 輪） |
|--------|------|------|----------------------|
| 5 人 | 3 | 2 | 2, 3, 2, 3, 3 |
| 6 人 | 4 | 2 | 2, 3, 4, 3, 4 |
| 7 人 | 4 | 3 | 2, 3, 3, 4\*, 4 |
| 8 人 | 5 | 3 | 3, 4, 4, 5\*, 5 |
| 9 人 | 6 | 3 | 3, 4, 4, 5\*, 5 |
| 10 人 | 6 | 4 | 3, 4, 4, 5\*, 5 |

> \* 第 4 輪（7 人以上）：需 **2 張失敗牌** 才算任務失敗

## 2. App 負責的功能（精簡版）

App 只做三件事：**夜晚語音主持**、**計分板**、**勝負判定**。

### 2.1 遊戲設定

組織者選擇：
1. **玩家人數**（5-10 人）
2. **角色分佈配置**（勾選要使用的角色）
   - 必選：梅林、刺客
   - 可選：派西維爾、莫甘娜、莫德雷德、奧伯倫
   - 剩餘自動填充好人騎士 / 邪惡爪牙
3. App 根據配置顯示「好人 N 人 / 邪惡 N 人 / 使用角色清單」
4. 組織者確認後，將實體角色牌洗牌發給玩家

### 2.2 夜晚語音主持

App 根據角色分佈配置，**自動播放對應的夜晚語音腳本**。這是 App 的核心價值。

語音流程會根據啟用的角色**動態組合**：

```
固定開場：「所有人閉上眼睛，夜晚降臨。」

→ [邪惡互認]（必有）
  - 無奧伯倫：「邪惡陣營，請睜開眼睛，確認彼此身份。」
  - 有奧伯倫：「邪惡陣營——奧伯倫除外——請睜開眼睛。」
  - 停頓 → 「邪惡陣營，請閉上眼睛。」

→ [梅林確認]（必有）
  - 「梅林，請睜開眼睛。」
  - 無莫德雷德：「以下是所有邪惡陣營成員，請確認。」
  - 有莫德雷德：「以下是邪惡陣營成員，但莫德雷德隱藏其中。」
  - 停頓 → 「梅林，請閉上眼睛。」

→ [派西維爾確認]（僅啟用派西維爾時）
  - 「派西維爾，請睜開眼睛。」
  - 有莫甘娜：「梅林與莫甘娜，請伸出拇指。」
  - 無莫甘娜：「梅林，請伸出拇指。」
  - 停頓 → 「派西維爾，請閉上眼睛。放下拇指。」

固定結束：「天亮了，所有人請睜開眼睛。」
```

### 2.3 計分板（任務進度）

夜晚結束後，App 進入**計分板模式**。螢幕顯示：

- 5 輪任務進度（成功 / 失敗 / 待進行）
- 每輪需要的任務人數
- 當前否決計數（0-4）
- 當前是第幾輪

**組織者唯一操作**：每輪任務結束後，回報任務結果：
- 點「任務成功」或「任務失敗」
- App 更新計分板 + 語音播報結果

**否決**：組織者點「提名被否決」，否決計數 +1。

### 2.4 勝負判定

App 自動判定：
- 好人 3 勝 → 語音播報 + 進入刺殺梅林階段
- 邪惡 3 勝 → 語音播報邪惡勝利
- 連續 5 次否決 → 語音播報邪惡勝利
- 刺殺梅林：組織者回報「刺殺成功」或「刺殺失敗」→ App 播報最終結果

## 3. 遊戲流程（App 視角）

```
[遊戲設定]
  選人數、選角色分佈
  ↓
[夜晚語音主持] ← App 核心功能
  TTS 自動播報閉眼/睜眼流程
  ↓
[計分板模式] ← 放桌上顯示
  ↓
  ┌─→ [等待組織者回報]
  │     │
  │     ├── 「任務成功」→ 成功 +1 → 語音播報
  │     ├── 「任務失敗」→ 失敗 +1 → 語音播報
  │     └── 「提名否決」→ 否決 +1 → 語音播報
  │     │
  │     ├── 成功 = 3 → [刺殺梅林階段]
  │     ├── 失敗 = 3 → [邪惡勝利]
  │     ├── 否決 = 5 → [邪惡勝利]
  │     └── 否則 → 繼續下一輪
  └─────┘
  ↓
[刺殺梅林]
  組織者回報：刺殺成功 / 失敗
  ↓
[結算]
  語音播報最終勝利方
  「再來一局」/「結束遊戲」
```

## 4. User Stories

### US-01：遊戲設定與角色分佈配置

**As a** 組織者
**I want** 選擇玩家人數並自訂角色分佈
**So that** 夜晚語音能播放正確的流程，且玩家知道要拿哪些實體角色牌

**Acceptance Criteria:**
- [ ] 選擇 5-10 人
- [ ] 勾選/取消可選角色（派西維爾、莫甘娜、莫德雷德、奧伯倫）
- [ ] 即時顯示角色分佈摘要：好人 N / 邪惡 N / 角色清單
- [ ] 角色總數 ≠ 玩家人數時，開始按鈕 disabled + 提示
- [ ] 根據人數提供推薦預設配置

**Priority:** P0

---

### US-02：夜晚語音主持

**As a** 所有玩家
**I want** App 自動播報夜晚閉眼/睜眼語音
**So that** 不需要真人主持夜晚流程，所有人都能參與遊戲

**Acceptance Criteria:**
- [ ] 語音腳本根據角色分佈動態組合（有/無奧伯倫、有/無派西維爾等）
- [ ] 每個步驟之間有適當停頓（可設定秒數）
- [ ] 播放中可暫停/恢復
- [ ] 播放中可跳過當前步驟
- [ ] 夜晚語音結束後自動進入計分板模式

**Priority:** P0

---

### US-03：計分板顯示

**As a** 所有玩家
**I want** 螢幕顯示任務進度和否決計數
**So that** 桌上放著就能看到遊戲狀態，不需靠記憶

**Acceptance Criteria:**
- [ ] 5 輪任務格子，標示成功（綠）/ 失敗（紅）/ 當前（金）/ 待進行（灰）
- [ ] 每格下方顯示該輪需要的任務人數
- [ ] 否決計數顯示（0-4 個圓點）
- [ ] 第 4 輪（7 人以上）標註「需 2 失敗」
- [ ] 放桌上遠距離可讀（大字體、高對比）

**Priority:** P0

---

### US-04：回報任務結果

**As a** 組織者
**I want** 點一個按鈕回報本輪任務結果
**So that** App 更新計分板並語音播報

**Acceptance Criteria:**
- [ ] 計分板底部三個按鈕：「任務成功」「任務失敗」「提名否決」
- [ ] 點擊後有確認（防誤觸）
- [ ] 確認後 App 語音播報結果
- [ ] 計分板即時更新
- [ ] 否決後否決計數 +1
- [ ] 任務執行後（成功或失敗）否決計數歸零

**Priority:** P0

---

### US-05：勝負判定與播報

**As a** 所有玩家
**I want** App 自動判斷勝負並語音播報
**So that** 遊戲結束時有儀式感

**Acceptance Criteria:**
- [ ] 好人 3 勝 → 「好人完成三次任務！但刺客還有最後機會...」
- [ ] 邪惡 3 勝 → 「邪惡陣營獲勝！」
- [ ] 否決 5 次 → 「連續五次否決，邪惡陣營直接獲勝！」
- [ ] 配合音效（勝利/失敗）

**Priority:** P0

---

### US-06：刺殺梅林階段

**As a** 組織者
**I want** 在好人三勝後，回報刺殺梅林的結果
**So that** App 判斷最終勝負

**Acceptance Criteria:**
- [ ] 好人 3 勝後自動進入刺殺階段
- [ ] 語音播報「刺客，請與同伴商議，指出梅林」
- [ ] 組織者回報：「刺殺成功」或「刺殺失敗」
- [ ] 成功 → 「梅林被刺殺，邪惡逆轉勝！」
- [ ] 失敗 → 「刺殺失敗，好人最終獲勝！」

**Priority:** P0

---

### US-07：再來一局

**As a** 組織者
**I want** 結算後快速重新開始
**So that** 不需要重新設定人數和角色配置

**Acceptance Criteria:**
- [ ] 「再來一局」保留人數和角色分佈設定
- [ ] 直接進入夜晚語音
- [ ] 「結束遊戲」回到遊戲選擇頁

**Priority:** P1

---

### US-08：語音與音效設定

**As a** 組織者
**I want** 調整語速、音量和夜晚步驟間隔
**So that** 適應不同場地環境和玩家偏好

**Acceptance Criteria:**
- [ ] 語速調整（0.5x - 2.0x）
- [ ] 語音/音效/背景音樂三個獨立音量
- [ ] 夜晚步驟間停頓秒數可調（3-10 秒）
- [ ] 設定儲存至 localStorage

**Priority:** P1

---

### US-09：規則語音說明

**As a** 新手玩家
**I want** App 播放阿瓦隆規則說明
**So that** 第一次玩也能快速了解規則

**Acceptance Criteria:**
- [ ] 設定頁有「規則說明」按鈕
- [ ] TTS 播報完整規則（約 1 分鐘）
- [ ] 播放中可跳過
- [ ] 也顯示文字版規則摘要

**Priority:** P2

---

### US-10：遊戲歷史記錄

**As a** 組織者
**I want** 自動記錄每場遊戲結果
**So that** 回顧過往對局

**Acceptance Criteria:**
- [ ] 遊戲結算後自動存入 IndexedDB

- [ ] 歷史頁顯示對局列表 + 簡易統計

**Priority:** P2

## 5. Edge Cases

### EC-01：第 4 輪特殊規則提醒

**情境**：7 人以上第 4 輪需要 2 張失敗牌才算失敗，玩家容易忘記。
**預期**：進入第 4 輪時，語音特別提醒 + 計分板標註。

### EC-02：否決計數歸零時機

**情境**：任務執行後（無論成敗）否決計數歸零，但連續否決不歸零。
**預期**：App 在「任務成功」或「任務失敗」時自動歸零；「提名否決」時只累加。

### EC-03：第 5 次否決

**情境**：否決達 4 次後，語音警告；達 5 次直接結束。
**預期**：第 4 次否決時加強警告語音；第 5 次直接判邪惡勝利。

### EC-04：組織者誤觸任務結果

**情境**：組織者不小心按錯「任務成功」。
**預期**：提供「撤銷上一步」按鈕（僅限最近一次操作，2 秒內可撤銷）。

### EC-05：夜晚語音被打斷

**情境**：來電或系統通知打斷 TTS 播放。
**預期**：播放被打斷後，暫停在當前步驟，組織者可點「重播本步驟」或「繼續下一步」。

### EC-06：好人三勝但沒有刺客

**情境**：理論上不應發生（刺客為必選），但做防禦性處理。
**預期**：若角色配置無刺客，好人三勝直接判好人勝利，跳過刺殺階段。

## 6. 夜晚語音腳本

### 開場

```
所有人閉上眼睛。夜晚降臨。
```

### 邪惡互認（無奧伯倫）

```
邪惡陣營成員，請睜開眼睛，確認彼此的身份。
（停頓 N 秒）
邪惡陣營，請閉上眼睛。
```

### 邪惡互認（有奧伯倫）

```
邪惡陣營——奧伯倫除外——請睜開眼睛，確認彼此的身份。
（停頓 N 秒）
邪惡陣營，請閉上眼睛。
```

### 梅林確認（無莫德雷德）

```
梅林，請睜開眼睛。
邪惡陣營成員，請伸出拇指。
（停頓 N 秒）
收回拇指。梅林，請閉上眼睛。
```

### 梅林確認（有莫德雷德）

```
梅林，請睜開眼睛。
邪惡陣營成員——莫德雷德除外——請伸出拇指。
（停頓 N 秒）
收回拇指。梅林，請閉上眼睛。
```

### 派西維爾確認（有莫甘娜）

```
派西維爾，請睜開眼睛。
梅林與莫甘娜，請伸出拇指。
（停頓 N 秒）
收回拇指。派西維爾，請閉上眼睛。
```

### 派西維爾確認（無莫甘娜）

```
派西維爾，請睜開眼睛。
梅林，請伸出拇指。
（停頓 N 秒）
收回拇指。派西維爾，請閉上眼睛。
```

### 結束

```
天亮了，所有人請睜開眼睛。
第一位隊長，請開始提名你的任務隊伍。
```

### 任務成功播報

```
任務結果揭曉——任務成功！
好人陣營取得第 N 次勝利。
```

### 任務失敗播報

```
任務結果揭曉——任務失敗！
邪惡陣營破壞了第 N 次任務。
```

### 否決播報

```
提名被否決。已連續否決 N 次。
```

### 第 4 次否決警告

```
警告——這是第四次連續否決。若下次提名再次被否決，邪惡陣營將直接獲勝！
```

### 第 5 次否決

```
連續五次否決！邪惡陣營不戰而勝！
```

### 刺殺梅林

```
好人陣營完成了三次任務！然而，黑暗尚未散去。
刺客，這是你最後的機會。與你的同伴商議，找出梅林。
```

### 刺殺成功

```
梅林被識破！邪惡陣營逆轉勝利！
黑暗，籠罩了卡美洛。
```

### 刺殺失敗

```
刺殺失敗！好人陣營最終獲勝！
亞瑟的騎士，守護了卡美洛的榮光。
```

### 邪惡三勝

```
三次任務失敗。邪惡陣營獲勝。
莫德雷德的計謀，成功顛覆了亞瑟的王國。
```

### 規則說明語音

```
歡迎來到阿瓦隆。
你們之中有人效忠亞瑟王，有人是莫德雷德的爪牙。
遊戲共有五輪任務。好人陣營需要完成三輪任務；邪惡陣營需要破壞三輪。
每一輪，隊長提名隊伍，全員公開投票。
投票通過，隊員秘密執行任務；被否決，下一位隊長接手。
連續五次否決，邪惡直接獲勝。
好人若完成三次任務，刺客將有最後一次機會——指出梅林。
若刺中，邪惡逆轉勝利。謹言慎行，命運取決於你們。
```
